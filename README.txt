
HƯỚNG DẪN UPLOAD GITHUB

1. Upload toàn bộ file lên GitHub

2. Vào:
Settings -> Pages

3. Chọn:
Deploy from a branch

4. Chọn:
main / root

5. Save

6. Web sẽ có dạng:
https://username.github.io/repository-name/

QUẢN LÝ TIN BÀI:
- Chỉ cần sửa file news.json
- Mỗi bài viết là 1 object JSON

VIDEO YOUTUBE:
- Chỉ cần thay youtube bằng ID video
Ví dụ:
https://youtube.com/watch?v=ysz5S6PUM-U

=> ID là:
ysz5S6PUM-U
